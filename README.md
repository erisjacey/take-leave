# TakeLeave.sg

Personal PTO tracking and balance forecasting for Singapore workers. Configure your leave accrual, log entries, and see a running balance chart — no account needed.

## Stack

- Next.js 16 (App Router) · TypeScript · Tailwind CSS v4
- Recharts · next-themes · localStorage (no backend)

## Development

```bash
npm install
npm run dev       # start dev server at localhost:3000
npm run check     # lint + format check
npm run fix       # lint fix + format write
npm run build     # production build
```
